This is a repo for my OS X dotfiles.
Bootstrap is based on the [dotbot](https://github.com/anishathalye/dotbot).   

## :green_apple: Installation

```sh
> git clone git@github.com:andrew-kandyba/dotfiles.git .dotfiles
> cd .dotfiles

> make run
```
